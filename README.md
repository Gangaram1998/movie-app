# Movie app
This Project is about a website through which we can search various movies, their poster, release date and rating.

# Tech Stack Used

  HTML <br/>
  CSS  <br/>
  JavaScript
  
 # Features
 Users can search any movie with the poster, rating and release date of the movie.
